THINGS TO WATCH
  - Halaal: NEVER mention wine, beer, spirits, pork, or cider in any copy.
  - pdfjs worker: /public/pdf.worker.min.mjs must stay in sync with
    node_modules/pdfjs-dist version. Don't upgrade pdfjs-dist without recoping.
  - Tailwind v4: no tailwind.config.js — all tokens are in app/globals.css @theme {}.
    Adding new colours/fonts = add to globals.css, not a config file.
  - Lucide React v1: no Facebook/Instagram/YouTube icons. Use inline SVGs (see
    FlipbookViewer.tsx IcoWA/IcoFB/IcoX for the pattern).
  - SSR and dynamic imports: any component that uses pdfjs or browser-only APIs
    must be wrapped in a "use client" file and imported with { ssr: false }.
    Example: CatalogueSection.tsx wraps FlipbookViewer.tsx.
